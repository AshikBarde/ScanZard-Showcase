# ScanZard - QR Code Attendance Management System

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge)](https://scanzard.com)

> **A modern QR code-based attendance tracking system built with React and Supabase.**

---

## 📋 Overview

ScanZard is a full-stack web application that streamlines employee attendance management using QR code technology. Employees scan a QR code to clock in/out, and administrators get real-time analytics and reports.

## ✨ Features

- **QR Code Scanning** — Employees scan QR codes to mark attendance instantly
- **Admin Dashboard** — Real-time analytics, attendance trends, and employee management
- **Employee Portal** — View personal attendance history and profile
- **Role-Based Access** — Secure authentication with admin and employee roles
- **Excel Export** — Download attendance reports for record-keeping
- **Responsive Design** — Works seamlessly on desktop and mobile devices
- **Offline Support** — PWA capabilities with offline fallback

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| **React 18** | Frontend UI framework |
| **Vite** | Build tool and dev server |
| **Supabase** | Backend (Auth, Database, Storage) |
| **Recharts** | Data visualization |
| **HTML5-QRCode** | QR code scanning library |
| **React Router** | Client-side routing |
| **Lucide React** | UI icons |
